### 如何部署本项目?

```python
# 安装npm (以ubuntu系统为例)
sudo apt -y install npm 

# 安装vue
npm install -g vue-cli

# 安装依赖
npm install

# 本地运行
npm run dev

# 编译本项目
npm run build
```

然后只需要将编译后的 `dist` 文件夹放到服务器上运行即可
